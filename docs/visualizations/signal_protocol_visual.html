🔧 Core Components

🔑 Key Exchange

  • X25519 (Curve25519 ECDH)
  • 256-bit security level
  • Forward secrecy via ephemeral keys
  • Based on discrete logarithm problem

🔄 Double Ratchet

  • Symmetric ratchet (chain key)
  • DH ratchet on each turn
  • Future secrecy on compromise
  • Out-of-order message handling

🔒 Encryption

  • AES-256-GCM
  • Authenticated encryption
  • Message key derivation
  • Padding for length hiding

✍️ Signatures

  • Ed25519 digital signatures
  • Pre-key bundle signing
  • Identity verification
  • Safety number comparison
⚠️

Quantum Vulnerability

Shor's Algorithm can break X25519 and Ed25519 on a quantum computer. Current messages captured today ("harvest now") could be decrypted in the future ("decrypt later") when quantum computers become available. Estimated timeline: 2030-2040.
